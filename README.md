# srp项目
## 依赖信息
目前最近版需要的依赖为 PyQt5 以及 PyQt-tool
可以使用如下命令安装

PyQt5:
```commandline
pip install PyQt5
```
PyQt5-tools:
```commandline
pip install PyQt5-tools
```

python-docx
```commandline
pip install python-docx
```

pywin32
```commandline
pip install pywin32
```

python-docx
```commandline
pip install python-docx
```


## 入口函数
本项目的入口函数为`main.py`

在所有依赖安装完之后便可以直接运行入口函数

可以直接使用如下命名运行入口函数
```commandline
python ./main.py
```
